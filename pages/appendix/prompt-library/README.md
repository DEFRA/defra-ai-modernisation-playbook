# Prompt Library

This library provides a selection of prompts used by authors. These examples serve as guides that you can adapt to your specific project needs, rather than using them verbatim.

## Best Practices
- Refer to [Prompting Guidance](prompting-guidance.md) for detailed guidance on writing effective prompts
- Tailor prompts to your project's specific requirements
- Iterate and refine prompts based on results

## Reusable prompt files for AI Coding Assistants

Some AI Coding Assistants support reusable prompt files that you can commit to your repository. These allow you to invoke standard prompts directly from your tool without copying and pasting.

For examples of reusable prompt files — including architectural decision records, test generation and security reviews — see the [Defra AI SDLC Tool Guidance](https://defra.github.io/ai-sdlc-tool-guidance/tool-setup/){:target="_blank"} site.
